## 실습 1: useEffect 연습 (30분)

### 목표
useEffect를 사용하여 데이터를 가져오고 관리합니다.

### 문제
다음 기능을 구현하세요:

1. **사용자 목록 가져오기**
   - useEffect를 사용하여 컴포넌트 마운트 시 사용자 목록 가져오기
   - 로딩 상태 표시
   - 에러 처리

2. **API 시뮬레이션**
   ```jsx
   // 가짜 API 함수
   const fetchUsers = async () => {
       await new Promise(resolve => setTimeout(resolve, 1000));
       return [
           { id: 1, name: 'John', email: 'john@example.com' },
           { id: 2, name: 'Jane', email: 'jane@example.com' }
       ];
   };
   ```

3. **사용자 상세 정보**
   - 사용자 클릭 시 상세 정보 표시
   - useEffect로 선택된 사용자 변경 시 데이터 가져오기

### 문제
다음 페이지를 가진 앱을 만드세요:

1. **페이지 구성**

3. **동적 라우팅**
   - 상품 ID를 URL 파라미터로 받기
   - 해당 상품 정보 표시

## 실습 4: 회원가입 폼 (40분)

### 목표
폼 검증을 포함한 회원가입 폼을 만듭니다.

### 문제
다음 기능을 가진 회원가입 폼을 만드세요:

1. **입력 필드**
   - 이름 (필수, 2자 이상)
   - 이메일 (필수, 이메일 형식)
   - 비밀번호 (필수, 8자 이상, 영문+숫자)
   - 비밀번호 확인 (필수, 비밀번호와 일치)
   - 전화번호 (선택, 숫자만)

2. **실시간 검증**
   - 입력 중 실시간으로 에러 메시지 표시
   - 유효한 필드는 초록색 테두리

3. **제출 처리**
   - 모든 필드가 유효할 때만 제출 가능
   - 제출 시 콘솔에 데이터 출력

### 제출물
- SignupForm.js 파일
- 실행 결과 스크린샷 (다양한 검증 상태)

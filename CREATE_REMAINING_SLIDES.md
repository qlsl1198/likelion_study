# 나머지 주차 HTML 슬라이드 생성 가이드

각 주차별로 `presentation.html` 파일을 생성해야 합니다.

## 생성 방법

각 주차의 `presentation.md` 파일을 기반으로 HTML 슬라이드를 생성합니다.

### 구조
1. 제목 슬라이드
2. 학습 목표
3. 각 섹션별 슬라이드
4. 핵심 정리
5. 과제
6. 참고 자료
7. 마지막 슬라이드

### 템플릿 사용
`week01/presentation.html` 또는 `week02/presentation.html`을 템플릿으로 사용하여
나머지 주차의 HTML 슬라이드를 생성할 수 있습니다.

### 주의사항
- 각 `<section>` 태그가 하나의 슬라이드입니다
- PDF 변환 시 각 `<section>`이 한 페이지로 나뉩니다
- 코드 블록은 `<pre><code class="언어">` 형식으로 작성합니다
- HTML 특수문자는 이스케이프 처리합니다 (`<` → `&lt;`, `>` → `&gt;`)

## 빠른 생성 방법

각 주차의 `presentation.md`를 읽어서 주요 섹션을 슬라이드로 변환합니다.

1. 제목과 학습 목표는 항상 포함
2. 각 주요 섹션(##)을 슬라이드로 변환
3. 하위 섹션(###)은 필요에 따라 별도 슬라이드 또는 같은 슬라이드에 포함
4. 코드 예제는 별도 슬라이드로 분리하는 것이 좋습니다

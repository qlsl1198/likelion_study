# 풀스택 개발 강의 커리큘럼 (React + Java Spring)

## 📚 강의 개요
- **기간**: 8주 (주차당 4시간)
- **방식**: PBL (Project-Based Learning)
- **기술 스택**: React (프론트엔드) + Java Spring Boot (백엔드)
- **목표**: 처음부터 끝까지 배울 수 있는 풀스택 개발자 양성

## 🎯 최종 프로젝트
**Todo 관리 웹 애플리케이션** - 사용자 인증, CRUD 기능, 실시간 업데이트를 포함한 완전한 웹 애플리케이션

## 📅 주차별 커리큘럼

### 1주차: 개발 환경 설정 및 웹 개발 기초
- 개발 환경 구축 (Node.js, Java, IDE)
- 웹 개발 기초 개념 (HTML, CSS, JavaScript)
- Git/GitHub 기초
- 프로젝트 구조 이해

### 2주차: React 기초
- React 개념 및 JSX
- 컴포넌트 기초
- Props와 State
- 이벤트 처리
- 실습: 간단한 카운터 앱

### 3주차: React 심화
- React Hooks (useState, useEffect, useContext)
- React Router
- 폼 처리 및 검증
- 실습: Todo 리스트 앱 (프론트엔드만)

### 4주차: Spring Boot 기초
- Spring Boot 소개 및 프로젝트 생성
- Spring MVC 패턴 이해
- RESTful API 설계
- 실습: 간단한 REST API 서버

### 5주차: Spring Boot 심화
- JPA/Hibernate를 이용한 데이터베이스 연동
- Spring Data JPA
- 엔티티 설계 및 관계 매핑
- 실습: Todo API 서버 (CRUD)

### 6주차: 프론트엔드-백엔드 연동
- CORS 설정
- Axios를 이용한 API 호출
- 에러 처리 및 로딩 상태 관리
- 실습: React + Spring Boot 통합

### 7주차: 인증/인가 및 고급 기능
- JWT 기반 인증 구현
- Spring Security 설정
- React에서 인증 상태 관리
- 실습: 사용자 인증 기능 추가

### 8주차: 배포 및 최종 프로젝트
- 빌드 및 최적화
- 배포 전략 (Vercel, AWS 등)
- 프로젝트 발표 및 코드 리뷰
- 실습: 최종 프로젝트 완성 및 배포

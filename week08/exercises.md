# 8주차 실습 문제

## 실습 1: 프로젝트 빌드 (30분)

### 목표
프로젝트를 프로덕션용으로 빌드합니다.

### 문제
다음 작업을 수행하세요:

1. **React 빌드**
   - `npm run build` 실행
   - 빌드 결과물 확인
   - 빌드 크기 확인

2. **Spring Boot 빌드**
   - Maven/Gradle 빌드 실행
   - JAR 파일 생성 확인
   - JAR 파일 실행 테스트

3. **환경 변수 설정**
   - 프로덕션용 환경 변수 파일 생성
   - 빌드 시 환경 변수 적용 확인

### 제출물
- 빌드 결과물 스크린샷
- 환경 변수 설정 파일

## 실습 2: 로컬 배포 테스트 (40분)

### 목표
로컬 환경에서 배포를 시뮬레이션합니다.

### 문제
다음 작업을 수행하세요:

1. **프론트엔드 서빙**
   - 빌드된 React 앱을 정적 서버로 서빙
   - Nginx 또는 serve 사용

2. **백엔드 실행**
   - 프로덕션 모드로 Spring Boot 실행
   - 프로덕션 데이터베이스 연결

3. **통합 테스트**
   - 프론트엔드에서 백엔드 API 호출 확인
   - 모든 기능 정상 작동 확인

### 제출물
- 실행 결과 스크린샷
- 설정 파일

## 실습 3: 클라우드 배포 (60분)

### 목표
실제 클라우드 플랫폼에 배포합니다.

다음 중 하나 이상을 선택하여 배포하세요:

1. **Vercel 배포 (프론트엔드)**
   - Vercel 계정 생성
   - GitHub 저장소 연결
   - 자동 배포 설정

2. **Railway 배포 (백엔드)**
   - Railway 계정 생성
   - 프로젝트 배포
   - 환경 변수 설정

3. **데이터베이스 설정**
   - 프로덕션 데이터베이스 생성
   - 연결 설정

### 제출물
- 배포된 URL
- 배포 과정 스크린샷
- 배포 설정 파일

## 종합 실습: 최종 프로젝트 완성 및 발표 (선택사항)

### 목표
8주 동안 배운 내용을 종합하여 완전한 프로젝트를 완성하고 발표합니다.

### 요구사항
1. **프로젝트 완성**
   - 모든 기능 구현
   - 에러 처리
   - 사용자 친화적 UI/UX

2. **배포**
   - 실제 배포 완료
   - 접근 가능한 URL 제공

3. **문서화**
   - README.md
   - API 문서
   - 배포 가이드

4. **발표 준비**
   - 발표 자료 (PPT 또는 마크다운)
   - 시연 준비
   - 질문 대비

### 발표 내용
1. 프로젝트 소개 (2분)
2. 주요 기능 시연 (3분)
3. 기술적 하이라이트 (2분)
4. 배운 점 및 향후 계획 (1분)
5. Q&A (2분)

### 제출물
- 전체 프로젝트 코드
- 배포된 URL
- 발표 자료
- 발표 영상 (선택사항)

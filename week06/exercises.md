## 실습 4: 에러 처리 개선 (30분)

### 목표
에러 처리를 개선하고 사용자 친화적인 메시지를 표시합니다.

### 문제
다음 에러 처리를 구현하세요:

1. **에러 타입별 처리**
   - 400: 잘못된 요청 메시지
   - 401: 인증 필요 메시지
   - 404: 리소스 없음 메시지
   - 500: 서버 오류 메시지
   - 네트워크 오류 메시지

2. **에러 표시**
   - 사용자 친화적인 메시지
   - 재시도 버튼

3. **Axios 인터셉터**
   - 응답 인터셉터로 에러 처리

### 제출물
- 에러 처리 유틸리티
- 수정된 컴포넌트
- 다양한 에러 상황 스크린샷

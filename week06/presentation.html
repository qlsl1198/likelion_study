1. CORS 설정

1.1 CORS란?

  • 정의: Cross-Origin Resource Sharing - 다른 출처(도메인) 리소스 접근 허용
  • Same-Origin Policy: 보안상 브라우저는 다른 출처 요청을 기본 차단
  • 문제: React(localhost:3000) → Spring(localhost:8080) 시 출처가 달라 차단됨
  • 해결: 서버에서 허용할 출처/메서드를 명시적으로 설정

1.2 Spring Boot CORS 설정

@Configuration
public class CorsConfig {
    @Bean
    public WebMvcConfigurer corsConfigurer() {
        return new WebMvcConfigurer() {
            @Override
            public void addCorsMappings(CorsRegistry registry) {
                registry.addMapping("/api/**")
                    .allowedOrigins("http://localhost:3000")
                    .allowedMethods("GET", "POST", "PUT", "DELETE", "PATCH")
                    .allowedHeaders("*")
                    .allowCredentials(true);
            }
        };
    }
}

2. Axios

2.1 Axios란?

  • 정의: HTTP 클라이언트 라이브러리
  • fetch vs Axios: fetch는 저수준. Axios는 인터셉터, 자동 JSON 변환, 에러 처리 편의
  • 설치: npm install axios
  • Promise 기반, 브라우저/Node.js 지원

2.2 기본 사용법

import axios from 'axios';

// GET 요청
axios.get('/api/todos')
    .then(response => console.log(response.data))
    .catch(error => console.error(error));

// POST 요청
axios.post('/api/todos', { title: 'New Todo', completed: false })
    .then(response => console.log(response.data));

2.3 Axios 인스턴스

const api = axios.create({
    baseURL: 'http://localhost:8080/api',
    timeout: 10000,
    headers: { 'Content-Type': 'application/json' }
});

api.get('/todos');
api.post('/todos', todoData);

3.1 커스텀 Hook (useTodos)

function useTodos() {
    const [todos, setTodos] = useState([]);
    const [loading, setLoading] = useState(true);
    
    useEffect(() => { fetchTodos(); }, []);
    
    const fetchTodos = async () => {
        const response = await api.get('/todos');
        setTodos(response.data);
        setLoading(false);
    };
    
    return { todos, loading, fetchTodos, createTodo, updateTodo, deleteTodo };
}

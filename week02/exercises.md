## 실습 1: 간단한 컴포넌트 만들기 (20분)

### 목표
기본적인 React 컴포넌트를 만들고 사용합니다.

### 문제
다음 요구사항을 만족하는 컴포넌트를 만드세요:

1. **Welcome 컴포넌트**
   - Props로 `name`을 받아서 "안녕하세요, {name}님!" 메시지 표시
   - 기본값: "Guest"

   - Props로 `text`와 `onClick` 함수를 받음
   - 클릭 시 onClick 함수 실행

3. **App 컴포넌트**
   - Welcome 컴포넌트 3번 사용 (다른 이름으로)
   - Button 컴포넌트 사용

### 목표
State를 사용하여 동적인 UI를 만듭니다.

### 문제
다음 기능을 가진 카운터 앱을 만드세요:

1. **기본 기능**
   - 현재 카운트 표시
   - 증가 버튼 (+1)
   - 감소 버튼 (-1)
   - 리셋 버튼 (0으로 초기화)

2. **추가 기능**
   - 카운트가 음수일 때 빨간색으로 표시
   - 카운트가 10 이상일 때 파란색으로 표시
   - 증가/감소 버튼에 +5, -5 버튼 추가

### 예시 코드 구조

```jsx
import { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    
    // 함수들 작성
    
    return (
        <div>
            {/* UI 작성 */}
        </div>
    );
}
```

## 실습 3: Todo 리스트 (프론트엔드만) (40분)

### 문제
다음 기능을 가진 Todo 리스트를 만드세요:

1. **기본 기능**
   - Todo 항목 추가 (입력 필드 + 추가 버튼)
   - Todo 항목 목록 표시
   - Todo 항목 삭제

2. **추가 기능**
   - Todo 항목 완료 표시 (체크박스)
   - 완료된 항목은 취소선 표시
   - 완료/미완료 필터링 (전체/완료/미완료)

### 데이터 구조 예시
```jsx
const [todos, setTodos] = useState([
    { id: 1, text: '리액트 공부하기', completed: false },
    { id: 2, text: '과제 제출하기', completed: true }
]);
```
